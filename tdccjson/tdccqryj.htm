📖 系統使用說明

啟動網頁時

  • 系統會先讀取瀏覽器 localStorage 內的 customStockGroups。
  • 若 customStockGroups 有資料，會依照群組 name 欄位由小到大排序，載入排序第一筆群組到文字輸入區，並自動產生股票表格。
  • 若 customStockGroups 沒有資料，才會使用舊流程讀取 savedStockList；若 savedStockList 也沒有資料，則保留頁面預設清單。
  • 自訂群組 Accordion 預設收起，展開後會看到已儲存群組，並依群組名稱排序顯示。

新增群組

  • 按下「增加群組」。
  • 在「群組名稱」輸入框填入群組名稱。
  • 在股票清單文字區輸入股票代號與名稱，每行一筆資料。
  • 按下「儲存 xxxxx 群組」後，資料會存入 customStockGroups，Accordion 內的群組清單會立即更新。

載入與修改群組

  • 展開「自訂群組」Accordion，按下群組名稱按鈕即可載入該群組內容。
  • 載入後系統會把群組內容放入文字輸入區，並自動執行「產生股票表格」。
  • 若載入 A 群組後只修改股票清單、群組名稱仍為 A，按下儲存會更新 A 群組。
  • 若載入 A 群組後把群組名稱改為 AB，按下儲存會保留 A 群組，並新增 AB 群組。
  • 若儲存時輸入的群組名稱已存在，系統會詢問是否覆蓋既有群組。

刪除群組

  • 展開「自訂群組」Accordion。
  • 按下群組旁的「刪除」按鈕。
  • 確認刪除後，該群組會從 customStockGroups 移除，Accordion 內的群組清單會立即更新。

股票表格與圖表

  • ⚠️ 重要提示：目前僅提供個股資料，不含 ETF。
  • 按下「產生股票表格」會把文字輸入區內容轉成雙排互動表格。
  • 點擊表格中的股票項目，會開啟 Chart.js 圖表，顯示散戶持股人數、大戶持股比例與週K線。
  • 圖表支援移動十字線，滑鼠移至資料點時會以 Tooltip 顯示詳細數據。
  • 週K線圖採用台灣慣例：紅漲、綠跌。

資料儲存位置與檢查方式

資料存在瀏覽器的 localStorage，也就是「使用者目前這個瀏覽器、這個網址來源」底下的本機儲存空間，不會寫成實體檔案。

目前使用到兩個 key：

savedStockList       // 舊版自訂股票清單，僅作為沒有 customStockGroups 時的 fallback
customStockGroups    // 目前自訂群組資料

檢查方式：

  • 用瀏覽器開啟 tdccqryj.htm。
  • 按 F12 開啟開發者工具。
  • 到 Application 或「應用程式」分頁。
  • 左側找 Local Storage (儲存空間)。
  • 點選「本機儲存空間」下方目前頁面的來源。
  • 找 savedStockList 與 customStockGroups。
